HttpServletResponse : 클라이언트에게 응답을 보내기 위한 기능들을 제공해주는 객체
	MIME Type : 서버와 브라우저간 통신이 발생할 때 본문으로 전달되는 데이터가 어떤 타입의 데이터 인지
				알려주기 위해 지정하는 헤더
	
	text(문자) : text/plain, text/html, text/css, text/javascript
	image(이미지) : image/gif, image/png, image/jpeg ...
	application(바이트) : application/octet-stream(이진 데이터)
	multipart(복합) : 복합, 바운더리를 사용해서 데이터들을 구분해준다.
